# Task 1: Data Cleaning and Preprocessing

**Internship:** Data Analyst Internship  
**Dataset:** Netflix Movies and TV Shows  
**Tools:** Python (Pandas)

## Steps Performed
1. Identified and handled missing values using `.isnull()` and `.fillna()`
2. Removed duplicate rows using `.drop_duplicates()`
3. Standardized text values for consistent formatting
4. Converted date columns into consistent format (`dd-mm-yyyy`)
5. Renamed column headers to lowercase and replaced spaces with underscores
6. Checked and corrected data types (integers, datetime)

## Files Included
- `netflix_titles.csv` → Raw dataset  
- `netflix_titles_cleaned.csv` → Cleaned dataset  
- `Task1_Data_Cleaning.ipynb` → Jupyter notebook with code  
- `README.md` → Summary of task

## Outcome
Created a clean, structured dataset ready for data analysis and visualization.
